## Git基本操作

-----

**1. git本地仓库操作**

+ 创建文件夹 `mkdir folder`
+ 创建和编辑文件 `vi file`
+ *VI* 基本操作有：`i插入 ` `Esc命令行键入模式` `:wq 保存后退出`
+ 文件更新到暂存区`git add file`
+ 将缓存区更新到存储区`git commit-m "massages"`
+ 查看日志`git log`
+ 本地提交到远程仓库 `git push origin master`
+ 其他回滚、分支管理、标签管理等操作下次再整理
